# Easyminal

#### The terminal for the average users

### --- ⚠️ Work in progress ---

## Technologies used

* [Tauri toolkit](https://tauri.app/)
* [Vue.js](https://vuejs.org/)
* [xterm.js](https://xtermjs.org/)

## Build instructions

### Requirements

* NodeJS
* `yarn`
* Rust (`cargo` + `rustc`)

### Build

```bash
# Install required modules
yarn install
```

```bash
# Run tauri startup script
yarn tauri dev
```
